5.什么是Lambda表达式？
    1.Lambda表达式也被称为箭头函数、匿名函数、闭包
    2.Lambda表达式体现的是轻级函数式编程思想
    3.'->'符号是Lambda表达式的核心操作符号，符号左侧是核心操作符号，符号右侧是操作表达式
    4.JDK8新特性


6.Model Code As Data
    1.Mode Code As Data，编码及数据，尽可能轻量级的将代码封装为数据。
    2.解决方案：接口&实现类(匿名内部类)
    3.存在问题：语法冗余、this关键字、变量捕获、数据控制等


7.项目问题：功能接口的设计及优化
    1.需求环境：线程类的创建。
    2.解决方案：匿名内部类实现。
    3.解决方案：Lambda表达式实现

8.为什么要使用Lambda表达式？
    1.他不是解决未知问题的新技术。
    2.对现有解决方案的语义优化。
    3.需要根据实际需求考虑性能问题。
